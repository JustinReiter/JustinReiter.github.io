Projects

Warlight.AI

Development of a smart bot, to be used for the Warzone game with the use of the Warlight API. Bot is developed with C# and implements various scoring and algorithms to pick and play similar to a higher end player. Code will evaluate expansion and picks based on location and possibilities for combinations and counters.

Decast

A project created in SE101 to convert a Raspberry PI into an infrared controller, with access to Amazon Web Services and controllable from Apple Devices. Built using C++ with use of LIRC for infrared interface and Amazon Lambda for communication between phone and Raspberry PI.

Western Blot Parser

A program built for use by a post-doctorate researcher to parse raw data from an excel file and outputs organized computations into separate excel file. Build in Java using Java Swing for the GUI and the Apache POI library for utilizing Excel files.

Tetris

A tetris game built in Java with the use of Java Swing for the GUI. Currently a work in progress.

Short/Cut

An application built during THacks2. Program optimizes use of raw material through progressively determining optimal places to cut. Built in Java, using Java Swing for the GUI and a Monte Carlo algorithm for optimizing cuts.

Genetic Algorithm - TSP

Implementation of a genetic algorithm to determine an optimized path for the travelling salesperson problem. Built in Java.
